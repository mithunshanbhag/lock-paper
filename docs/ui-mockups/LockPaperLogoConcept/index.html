Brand concept

LockPaper

A picture-first mark built from layered wallpaper cards, so the icon reads as photos and rotating backgrounds before anything else.

Why it fits

The stacked photo cards make the wallpaper story immediate, while the folded corner keeps the “paper” part of LockPaper literal without leaning on a lock glyph.

Visual language

The sky, landscape, and warm sun accent keep it picture-led, while the blue-cyan-gold palette still matches the app's existing UI colors.
